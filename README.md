# осторожнее эпилептикам, потому что это консольное приложение и здесь будет морграть всё при выводе

# программа, которая проходит лабиринт, с помощью алгоритма bfs(поиск в ширину) и генерирует этот лабиринт, с помощью алгоритма dfs(поиск в глубину)

текст из презентации:

**Язык программирования: C++.**

**Использование символов для отрисовки ( # — стены, * — самый короткий путь).**

**Возможность выбора между быстрой генерацией и режимом «наблюдения» за работой алгоритмов.**

**Алгоритм работает быстро даже на максимальных размерах поля (O(N⋅M)). Если пользователь выбрал, чтобы лабиринт выводился постоянно, то O((N*M)²)**
